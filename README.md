# webserver
This is a simple web server based on Flask 

# To run this websever  locally
Install python3 on local host or virual env
`pip install -r requirements.txt`
# To run the app
`python3 app2.py`

`http://0.0.0.0:4080`

`docker  build -t flaskapp  .`
